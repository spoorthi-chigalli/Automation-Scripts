# Selenium WebDriver Automation Practice

## Overview

This repository contains Selenium WebDriver automation scripts developed using Java. The project covers various Selenium concepts, web element interactions, browser handling techniques, and real-world automation scenarios.

## Technologies Used

* Java
* Selenium WebDriver
* Eclipse IDE
* ChromeDriver

## Topics Covered

### Web Elements

* Click, Clear, SendKeys
* isDisplayed()
* isEnabled()
* isSelected()
* getLocation()
* getSize()
* getRect()

### Locators

* ID Locator
* Name Locator
* Class Name Locator
* Tag Name Locator
* LinkText Locator
* PartialLinktext Locator
* CSS Locator
* xpath Locator
* Relative Xpath

### Dropdown Handling

* Single Select Dropdown
* Multi Select Dropdown
* Dropdown Assignments

### Mouse Actions

* Drag and Drop
* Right Click Actions
* Scroll
* Click Action
* HoldOn Action
* MoveTo Action
* Mouse Hovering

### Popups Handling

* Alert Popups
* Authentication Popups
* File Upload Popups
* Notification Popups
* Calendar Popups
* Print Popups
* Multiple Windows Handling
* Multiple Tabs Handling

### Multiple Elements Handling

* Printing All Links
* Auto Suggestions
* Checkbox Selection
* Web Table Handling

### Synchronization
* Implicit Wait
* Explicit Wait
* Fluent Wait
* Custom Wait
  
## Project Objective

The objective of this project is to gain hands-on experience in Selenium WebDriver automation and understand different web automation techniques used in real-world testing projects.
